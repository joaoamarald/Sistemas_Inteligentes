Tipos de Inteligência Artificial

IA Generativa:A IA generativa se concentra em criar conteúdo novo e original, como texto, imagens, códigos, etc. Esse tipo de IA utiliza grandes volumes de dados aprendidos pela mesma para gerar resultados próximo ao de humanos. Ex: ChatGPT, DALL-E.

LLM (Large Language Models): LLM (Large Language Models): São sistemas que foram treinados com grandes quantidades textuais em sua maioria para entender, gerar, e manipular a forma humana de escrever e expressar de forma natural e contextual. Ex: GPT, BERT.

Visão Computacional: A visão computacional permite a sistemas que "vejam" e interpretem vídeo, imagens e outros tipos de coisas audiovisuais. Utilizada para que os sistemas captem as características visuais de uma imagem por exemplo, para entender oque está naquela imagem e oque ela representa, criando uma alusão a visão humana. Ex: reconhecimento facial, carros autônomos.

Bibliotecas em Python para IA

Keras: É um API(Interface de programação de aplicações), utilizado para construir e treinar modelos de deep learning, criada pra ser uma biblioteca intuitiva e flexível para permitir que você prototipe rede neurais rapidamente. Hoje em dia ela é integrada ao TensorFlow(framework de aprendizado de máquina open-source do Google).

PyTorch: É uma biblioteca open-source de ML criada pela Meta, diferente da TensorFlor, o Pytorch usa um gráfico de computação dinâmico e isso permite que o código se assemelhe mais ao Python puro e seja mais fácil de depurar, graças a essa semelhança torna mais fácil experimentações em tempo real.

Scikit-Learn: : é uma biblioteca open-source de ML para Python diferente das Keras e Pytorch que focam no deep learning, o Scikit-Learn foca em algoritmos clássicos de ML, isso permite executar ações como classificação, regressão, clustering, redução de dimensionalidade e pré-processamento de dados. Hoje integra o ecossistema SciPy.

Bibliotecas de Visão Computacional

OpenCV: É uma biblioteca open-source especializada em processamentos de imagens e visão computacional. Foi criada inicialmente pela Intel em 1999, atualmente mantida por uma comunidade global, utilizada principalmente em robótica, vigilância e análise de imagens. Apesar de utilizar outras linguagem como C++ e Java, Python ainda é a mais utilizada dentro do OpenCV.

YOLO (You Only Look Once): É um algoritmo de detecção de objetos em tempo real. O destaque da YOLO é que em vez de analisar regiões de interesse em múltiplas etapas em uma imagem por exemplo, ele divide a imagem em uma grade e prevê simultaneamente as caixas delimitadoras.

MediaPipe: É um framework open-source desenvolvido pelo Google para construir pipelines de ML especializadas em processar dados multimídia como vídeos, imagens e áudios. O MediaPipe oferece soluções prontas pra uso para facilitar que os desenvolvedores criem, personalizam e implantem modelos de IA para executar tarefas como reconhecimento facial rastreamento de mãos, entre outros.
